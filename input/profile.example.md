# Candidate Profile

Copy this file to `input/profile.local.md` and replace it with a concise text export of your CV.

Recommended sections:

- Target roles
- Core skills
- Work history
- Selected achievements
- Location/timezone constraints
- Salary floor
- Dealbreakers

The bot does not need the original PDF for MVP ranking. A text or Markdown CV avoids
adding heavy PDF parsing dependencies and keeps Docker cheap.
